# ECE4010J Midterm Example Pack

This folder contains exam-style probability examples based on the scope in cheating_sheet.md.

## Files

| File | Use |
|---|---|
| all_examples.md | Main combined example pack. Covers PMF, density, CDF, distributions, approximations, Poisson process, transformations, joint distributions, conditional expectation, covariance, reliability, hazard, Weibull, and MGF. |
| 01_basics.md | Small quick-start file for PMF, PDF/CDF, expectation, variance, and LOTUS. |

## Suggested use

1. Read all_examples.md as the main printable reference.
2. Use 01_basics.md for the first-page quick review.
3. For exam problems, always do: identify distribution, write support, choose sum or integral, then compute.

## Notes

The original local pack had separate files per topic. The repository version is combined into one readable file to make GitHub writing stable and easy to print.
