# Basic Probability Examples

Example 1: normalize a discrete distribution.

A random variable takes values 0, 1, 2, 3 with probabilities proportional to x+1. The constant is 1/10. Then the probabilities are 0.1, 0.2, 0.3, 0.4. The cumulative probability up to 2 is 0.6. The mean is 2 and the variance is 1.

Example 2: normalize a continuous density on interval 0 to 2.

For density c times x squared, the constant is 3/8. The cumulative distribution inside the interval is x cubed over 8. The probability between 1 and 2 is 7/8.
